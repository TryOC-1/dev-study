# Airflow Study

## Installation
- 가상환경 세팅
```python
python -m venv venv
```
- 파이썬 패키지 설치
```python
pip install -r requirements.txt
```
- pre-commit 설치
```bash
pre-commit install
```

## Description
- Airflow 초기 버전에 대한 분석부터 시작 (1.x.x)
- fork로 자신의 repository 생성 및 커밋 관리
- Issue를 통해 과제 생성
- Pull requests를 통해 해당 과제에 대한 제출
- 주단위로 디렉터리 생성 및 과제 내용 커밋
